## 🔬 实验复习重点 (Key Takeaways)

### Work 1: 单相桥式全控整流 (阻感负载)
* **大电感条件**：当 $\omega L \gg R$ 时，负载电流 $i_d$ 连续且近似恒定。
* **导通角特性**：在电流连续状态下，晶闸管导通角 $\theta$ 恒为 $180^\circ$，与触发角 $\alpha$ 无关。
* **移相范围**：整流状态下的 $\alpha$ 移相范围为 $0^\circ \sim 90^\circ$。当 $\alpha=90^\circ$ 时，$U_d=0$。

### Work 2: 三相桥式全控整流 (电阻/阻感负载)
* **负载对比**：电阻负载在 $\alpha > 60^\circ$ 时波形断续；阻感负载依靠大电感续流，波形在 $\alpha > 0^\circ$ 时即出现负电压面积。
* **误差分析**：由于 Simulink `Universal Bridge` 模块考虑了晶闸管的导通电阻 ($R_{on}$) 和正向压降 ($V_f$)，仿真测得的 $U_d$ 会略低于教材理想公式计算值。

## 💻 运行说明
1. **软件要求**：MATLAB R2023b 及以上版本。
2. **模块库**：需安装 Simscape Electrical 工具箱。
3. **操作**：
   * 打开 `Models/` 下的 `.slx` 文件。
   * 双击 `Constant` 模块修改 `Alpha`（控制角）数值。
   * 点击运行，通过 `Scope` 观察负载端波形。
